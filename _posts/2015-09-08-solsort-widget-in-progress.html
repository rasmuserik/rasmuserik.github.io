Next I am working on implementing custom widgets. This is in the sense of a piece of code that is embedded in a website to add certain features. The core embedding could be implemented with these 2 steps:

  • Load the code, ie. add a <script> tag.
  • Add the widget code in the html, it <div class=solsort-widget data-somekey="some-value"></div>.

And then all of the elements with the solsort-widget class will be transformed into actual widgets based on the data-parameters.
